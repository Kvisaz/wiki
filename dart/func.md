# Функции  в Dart

## Литерал функции
 как в JavaScript, за исключением того, что var
```dart 
var incAge = (age) => age + 1;
```

## Аргументы по умолчанию
 передаются в фигурных скобках (curly braces)
```dart
    var greet = ({ name = 'Rocky' }) => "Woof! My name is ${name}";
```
