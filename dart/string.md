# Строки в Dart

Можно применять как одинарные, так и двойные кавычки. 

Строковый шаблон - как в TypeScript. Но если не выражение, а идентификатор - можно опускать фигурные кавычки. 

```dart
    var name = 'Vanya'
    var greet = "Woof! My name is ${name}";
    var greet = "Woof! My name is $name";
```
